## Трекер задач на C++ с использованием библиотеки Qt

# Возможности
- добавление задач
- редактирование
- отметка выполненных
- удаление
- хранение списка задач

# Реализовано:
- окно ввода задачи
- список задач
- добавление через кнопку
- добавление через Enter
- фокус на задаче через ЛКМ
- удаление выделенной задачи через кнопку
- редактирование задачи из списка
- чек-бокс задачи
- смена шрифта задачи на зачеркнутый
- хранение списка задач между сеансами
- изменена стилистика кнопок, шрифтов
